# HouseFlow48

HouseFlow48 es una aplicación privada de control financiero doméstico orientada a disciplina y claridad de cashflow mensual.

No es contabilidad formal.  
No es banca digital.  
Es un sistema de control conductual financiero para un núcleo familiar.

El foco emocional del producto es:

Control compartido + disciplina financiera + claridad absoluta del flujo mensual.
